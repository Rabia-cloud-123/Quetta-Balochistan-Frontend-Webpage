# Quetta Balochistan Website (Frontend)

This repository contains the **frontend part** of the Quetta Balochistan website.

## Folder Structure

```
frontend/
│
├── Show.html     # Main HTML page
├── style.css      # CSS styling
└── script.js      # JavaScript for API calls and interaction
```

## Features

* Displays famous places in Quetta City
* Displays traditional Balochi foods
* Contact form to send messages
* Fetches data from backend API (server.js)

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Rabia-cloud-123 /quetta-website.git
```

2. Navigate to the frontend folder:

```bash
cd quetta-website/frontend


## Notes

* This is only the frontend. 
* No JAR file is needed as this is a Node.js + HTML project.
* You can modify `script.js` to change API endpoints or display logic.
